# Cursorから直接フォルダを開く方法

## 🎯 Cursorのファイルエクスプローラーから開く

### 方法1: Cursorのファイルエクスプローラーから

1. **Cursorの左側のファイルエクスプローラーを見る**
   - 現在開いているファイルが表示されています

2. **プロジェクトのルートフォルダを探す**
   - `study-abroad-comparison` フォルダを探す
   - または、`package.json` や `app` フォルダがある場所を探す

3. **フォルダを右クリック**
   - 「Reveal in File Explorer」を選択
   - または「フォルダーをエクスプローラーで表示」
   - これでエクスプローラーが開きます

---

### 方法2: 現在開いているファイルから

1. **現在開いているファイル（例：`ZIP_PROJECT_GUIDE.md`）を確認**
   - ファイルパスが表示されています：`c:\Users\kiris\.cursor\study-abroad-comparison\ZIP_PROJECT_GUIDE.md`

2. **ファイルを右クリック**
   - 「Reveal in File Explorer」を選択

3. **エクスプローラーでフォルダが開く**

---

### 方法3: Cursorのメニューから

1. **Cursorのメニューバーで「File」をクリック**
2. **「Open Folder...」または「フォルダーを開く」を選択**
3. **既に開いているフォルダの場所が表示される**
   - そこからフォルダを探す

---

## 📦 ZIP化する

エクスプローラーでフォルダが開けたら：

1. **フォルダをZIP化**
   - フォルダを右クリック → 「送る」→ 「圧縮（zip形式）フォルダー」
   - または、フォルダ内のファイルをすべて選択（`Ctrl + A`）
   - 右クリック → 「送る」→ 「圧縮（zip形式）フォルダー」

---

## 🔍 確認方法

現在開いているファイルから、フォルダの場所を確認できます：

- ファイルパス：`c:\Users\kiris\.cursor\study-abroad-comparison\ZIP_PROJECT_GUIDE.md`
- つまり、フォルダは：`c:\Users\kiris\.cursor\study-abroad-comparison`

---

**Cursorのファイルエクスプローラーで、`study-abroad-comparison`フォルダを右クリックして「Reveal in File Explorer」を試してみてください！**
