# GitHub Desktopで解決する方法（最も簡単）

## 🎯 なぜGitHub Desktop？

- Gitコマンドが不要
- ZIP化が不要
- エラーが少ない
- 視覚的に操作できる

---

## 📥 ステップ1: GitHub Desktopをダウンロード

1. **ブラウザで以下を開く**
   - https://desktop.github.com/

2. **「Download for Windows」をクリック**

3. **インストール**
   - ダウンロードしたファイルを実行
   - インストールウィザードに従う

4. **GitHub Desktopを起動**
   - GitHubアカウントでログイン

---

## 📂 ステップ2: ローカルリポジトリを追加

1. **GitHub Desktopを開く**

2. **「File」→「Add local repository」をクリック**

3. **リポジトリのパスを選択**
   - 「Choose...」をクリック
   - `C:\Users\kiris\.cursor` を選択
   - 「Add repository」をクリック

---

## ✅ ステップ3: 変更をコミット

1. **変更されたファイルが表示されます**

2. **コミットメッセージを入力**
   - 例：`Initial commit: Sappy Study Abroad Platform`

3. **「Commit to main」をクリック**

---

## 🚀 ステップ4: GitHubに公開

1. **「Publish repository」ボタンをクリック**
   - または、「Branch」→「Publish branch」

2. **リポジトリ情報を入力**
   - Name: `sappy-study-abroad`（好きな名前）
   - Description: （空欄でもOK）
   - ✅ 「Keep this code private」を外す（Publicにする場合）

3. **「Publish repository」をクリック**

---

## ✅ 完了！

これで、プロジェクトがGitHubにアップロードされました！

---

## 📝 今後の更新方法

コードを変更したら：

1. **GitHub Desktopを開く**
2. **変更されたファイルを確認**
3. **コミットメッセージを入力**
4. **「Commit to main」をクリック**
5. **「Push origin」をクリック**

---

**GitHub Desktopを使えば、Gitコマンドで悩む必要がありません！**
