# Pull (Rebase)を試す方法

## 🎯 Pull (Rebase)とは

通常のPullはマージコミットを作成しますが、Rebaseを使うと、よりクリーンな履歴になります。

---

## ✅ 手順

### ステップ1: Pull (Rebase)を実行

1. **コマンドパレットを開く**
   - `Ctrl + Shift + P` を押す

2. **Pull (Rebase)を検索**
   - 「**pull rebase**」と入力
   - または「**Git: Pull (Rebase)**」と入力

3. **選択して実行**
   - 「Git: Pull (Rebase)」を選択

### ステップ2: 認証が求められたら

- **ユーザー名**: `kirishio1019-jpg`
- **パスワード**: パーソナルアクセストークン

### ステップ3: 完了後、Push

Rebaseが完了したら：
- `Ctrl + Shift + P` → 「**Git: Push**」

---

## 🆘 エラーが出た場合

Rebaseでもエラーが出る場合、選択肢が表示されることがあります。その場合は：

1. **表示された選択肢を確認**
2. **適切なものを選択**
3. **または、GitHub Desktopを使用**

---

**まずは「Git: Pull (Rebase)」を試してみてください！**
